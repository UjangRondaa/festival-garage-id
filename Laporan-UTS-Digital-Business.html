2. BAGIAN 1 — COPYWRITING & LANDING PAGE CPMK04 · 40%

2.1 Konsep Copywriting

Copywriting Festival Garage ID dirancang dengan prinsip jelas, persuasif, dan berorientasi konversi. Bahasa yang digunakan adalah bahasa kasual khas komunitas gamer Indonesia — bersahabat namun profesional, sesuai dengan profil target audiens (usia 17–35 tahun, casual gamer hingga hardcore collector FH5).

Headline Utama

"Mobil Langka FH5 Langsung Masuk Garasi. Tanpa Grinding."

Headline ini langsung memukul pain point utama pemain FH5: grinding berjam-jam untuk mendapatkan satu mobil eksklusif. Kata kunci "Tanpa Grinding" diberi penekanan visual (warna oranye gradient) sebagai janji utama brand.

Sub-headline

"Festival Garage ID — diler virtual terpercaya untuk mobil unicorn, ultra-rare, & Festival Playlist exclusive. Transfer aman via Auction House, dipandu tim berpengalaman."

Call to Action (CTA)

  • CTA Utama: "Chat WhatsApp Sekarang" → langsung diarahkan ke wa.me dengan pesan template.
  • CTA Sekunder: "Lihat Katalog" → smooth scroll ke section katalog ready stock.
  • CTA Final: "Hubungi Admin Sekarang" di section penutup, dengan badge "Admin Online · Respon < 5 Menit".

3.4 Langkah-Langkah Konfigurasi (Step-by-Step)

Langkah 1 — Setup Lokal

# Buat folder project & inisialisasi Git
mkdir -p ~/Documents/festival-garage-id
cd ~/Documents/festival-garage-id
git init

Langkah 2 — Buat File HTML / CSS / JS

File index.html, style.css, dan script.js dibuat sesuai struktur 6 section yang dibahas di Bagian 1. Brand color konsisten: oranye #E8490F + dark background.

Langkah 3 — Commit ke Git

git add .
git commit -m "Initial commit: Festival Garage ID landing page"
git branch -M main

Langkah 4 — Buat Repository di GitHub

  1. Login ke github.com.
  2. Klik tombol "+" pojok kanan atas → "New repository".
  3. Repository name: festival-garage-id
  4. Visibility: Public (wajib agar bisa pakai GitHub Pages gratis).
  5. Jangan centang "Add README" (file sudah ada di local).
  6. Klik "Create repository".

Langkah 5 — Push ke GitHub

git remote add origin https://github.com/USERNAME/festival-garage-id.git
git push -u origin main

Langkah 6 — Aktifkan GitHub Pages

  1. Buka repo di GitHub → tab Settings.
  2. Sidebar kiri → Pages.
  3. Section "Build and deployment":
    • Source: Deploy from a branch
    • Branch: main · folder / (root)
  4. Klik Save.
  5. Tunggu 1–2 menit. GitHub akan menampilkan URL live di atas.
